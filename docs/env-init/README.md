### 初始化

新环境初始化，运行命令

```
xmmp env config
```

初始化得到 `xmmprc.json` 配置文件，需要配置环境初始化必要的字段

```
xmmp env init
```

会根据配置文件执行初始化操作，要求输入管理员账号登录后操作！


### 包的覆盖

运行

```
xmmp env cover
```

会根据 `xmmprc.json` 中的配置来读取并覆盖
